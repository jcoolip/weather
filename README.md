
# :rocket: Job Market Analyzer

What started as an attempt to compile the highest number of jobs related to my new career path, using the skills I've solidified along the way, turned into an all out data collecting venture. 

I started digging into the API's and RSS feeds that would fuel the data hoarding to come. Part of the excitement was normalizing the vast amount of information that these sources provide. Every source has a different way of storing or serving their data. Once a sort of "this is what I need" template emerged from the chaos, the data began to take shape.
